# Open Internet Summer 2024

Welcome to the Open Internet Summer (OIS) platform for registering participants and storing their submissions.

##

## Dev Instructions

# Initial setup

Install node, mops, dfx.

- `npm install`
- `mops install`

# Daily setup

- (new tab) `dfx start`
- (previous tab) `dfx deploy` or `npm run dev`
